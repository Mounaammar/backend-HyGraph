# Input Files Directory

This directory is for storing input data files for HyGraph ingestion.

## Expected Structure

```
inputFiles/
├── json/
│   ├── nodes/
│   │   └── Station.json
│   └── edges/
│       └── Trip.json
└── csv/
    ├── nodes/
    │   └── Station.csv
    ├── edges/
    │   └── Trip.csv
    └── measurements.csv
```
